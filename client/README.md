#### Установка зависимостей

```bash
yarn
```

или

```bash
npm install
```

---

#### Генерирование типов по GraphQL схеме сервера

Необходимо, чтобы GraphQL сервер был запущен!

```bash
yarn generate
```

или

```bash
npm run generate
```

---

#### Запуск в режиме разработки

```bash
yarn start
```

или

```bash
npm run start
```

---

#### Сборка проекта

```bash
yarn build
```

или

```bash
npm run build
```
